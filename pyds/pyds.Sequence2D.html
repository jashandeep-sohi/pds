pyds.Sequence2D¶

class pyds.Sequence2D(*values)¶

Bases: pyds.Sequence1D

Represents a 2D PDS sequence value.

Methods

Sequence2D.insert(index, value)¶

Insert value value at index index.

value must be an instance of Sequence1D, otherwise raise TypeError.

Special Methods

Sequence2D.__getitem__(index)¶

Return value at index index.

Sequence2D.__setitem__(index, value)¶

Note

See insert().

Sequence2D.__delitem__(index)¶

Remove the value at index index.

Sequence2D.__len__()¶

Return the number of values in the sequence.

Sequence2D.__iter__()¶

Return an iterator that iterates over the values in the sequence.

Sequence2D.__str__()¶

Return a PDS serialized string (str) representing the object.

Called by str().

Raises

  • RuntimeError

    If the sequence does not contain at least 1 value.
