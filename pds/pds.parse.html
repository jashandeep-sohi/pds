pds.parse¶

pds.parse(byte_string)¶

Parse byte_string into a Label object.

Parameters:

byte_string (bytes or mmap.mmap)

A valid PDS label string.
Raises:

ParsingError

If byte_string is not a valid PDS string.
Returns:

Label
Examples:

To parse a PDS label attached to a file:

>>> import pds
>>> file_object = open("./data/test.img", "r+b")
>>> file_content = file_object.read()
>>> pds.parse(file_content)
<pds.Label object at 0x...>

Rather than reading the entire contents of the file into a variable and then parsing it, you could use a mmap.mmap object. This is especially useful if the file is large:

>>> import mmap
>>> file_mmap = mmap.mmap(file_object.fileno(), 0)
>>> pds.parse(file_mmap)
<pds.Label object at 0x...>

The bytes representation of a Label object is a valid PDS string which when parsed leads to an identical Label object:

>>> label = pds.parse(file_mmap)
>>> bytes(label) == bytes(pds.parse(bytes(label)))
True
